### T-money卡
* 帶有“One Card All Pass”標識，全國通用。
* Tmoney卡/悠遊卡](https://www.kkday.com/zh-tw/product/18161-t-money-public-transit-card-pick-up-at-taiwan-taoyuan-international-airport-south-korea?cid=2810)，KKDAY買無充值。
* 1日、2日、3日卡:SEOUL CITYPASS。分為1日卡、2日卡、3日卡3種類型， 一天可自由乘坐地鐵和市區巴士的次數多達20次。這種T-money卡僅限於在首爾首都圈內使用， 乘坐首爾城市觀光遊巴士時，利用這種T-money卡可在市中心、古宮路線上的任一地點免費乘坐。價格：1日券15,000韓元，2日券25,000韓元，3日券35,000韓元。購買地點：全家、迷你島、首爾市內部分旅遊諮詢處、韓國智能卡公司
* M-PASS: 可根據需購買1日、2日、3日、5日、7日卡。 憑此卡可乘坐首都圈地鐵及鐵路、首爾市區巴士的幹線(藍色)巴士、支線(綠色)巴士和機場巴士， 在卡所限使用的天數內可自由乘坐上述線路列車，一天最多可使用20次。另外，它也具有T-money卡的功能，在地鐵站可當交通卡刷卡支付。購買這種卡時，還能得到中文宣傳冊、卡套及相關餐廳等的優惠券。價格：1日券10,000韓元，2日券18,000韓元，3日券25,500韓元，5日券42,500韓元，7日券59,000韓元※購買時需加付押金(5,000韓元)，使用後可前往購買處或機場內的GS25退卡，退卡時可拿回除去手續費500韓元以外的4,500韓元押金。購買地點：仁川機場1層的首爾市觀光諮詢中心,首爾市內的明洞諮詢觀光中
* KOREA PASS:這是在首爾及釜山等主要城市都能使用的充值卡。在N首爾塔、63大廈、樂天世界、樂天免稅店等60多處觀光地、商店、娛樂場所等均可享受優惠。根據預存金額分為4種類型。額外多付10,000韓元可選擇開通交通卡功能。這10,000韓元同時被存入交通卡內。以後可以取出餘額(有時需支付手續費)。價格：50,000韓元，100,000韓元，300,000韓元，500,000韓元※加付10,000韓元就可用作交通卡購買地點：仁川國際機場韓國鐵道公社服務臺、金浦國際機場國際線服務臺、機場鐵路A’REX售票處(仁川站、金浦站、首爾站)、釜山巴士客運站旅遊諮詢處、高速列車KTX釜山站樂天會員諮詢中心、韓國旅遊發展局、首爾和釜山7-11便利店及樂天瑪特(也有部分店鋪不提供銷售)

##求禮郡山茱萸村
地址 全羅南道求礼郡山洞面位安里
位於海拔 400 米處的求禮郡山茱萸村每到 3 月份就會被漫山遍野的鮮黃色山茱萸樹覆蓋。 為了迎接這一盛景，相關部門自 1999 年起開始舉辦盛大的山茱萸慶典活動，慕名前來的遊客絡繹不絕。 全羅道以各種美食聞名韓國，慶典期間不僅可以品嚐到全羅道的特色飲食和傳統酒品，還能欣賞音樂會等演出並參與豐富多元的體驗參與活動。

而且附近還有被列為韓國第一號國立公園的智異山（1,915 米）。 無論是登山新手還是專業人士，都可以不費力地攀登到智異山的山頂「天王峰」。 站在天王峰俯瞰足下，一幅壯麗的美景畫卷即在眼前鋪陳開來。 智異山的溫泉浴可以為遊客掃去登山的疲憊之感，如若是夏季，遊客還可以在智異山清澈的溪谷中戲水游玩。
